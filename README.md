# Spylt Landing Page

## Overview

**Spylt Landing Page** is a modern, highly interactive product landing page inspired by the **Awwwards Site of the Day** design style. The project focuses on delivering premium user experience through immersive scroll animations, smooth transitions, and responsive layouts.

The landing page showcases a fictional beverage brand with cinematic storytelling, animated typography, parallax effects, and engaging visual interactions built using **GSAP**.
